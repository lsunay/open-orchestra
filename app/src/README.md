# App Source

This directory contains the Solid.js control panel UI for OpenCode Boomerang.

## Structure

- `components/` - UI components, dialogs, layout, and shared widgets.
- `context/` - application state providers (OpenCode, layout, skills, db).
- `pages/` - route-level pages (chat, dashboard, onboarding, settings).
- `lib/` - shared utilities and constants.
- `types/` - shared frontend types.
